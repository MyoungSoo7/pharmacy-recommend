# 기능 명세서 — 약국 추천 시스템

## 1. 주소 검색

### 1.1 카카오 지오코딩 API 연동
- 사용자가 입력한 주소를 카카오 API로 전송
- 위도/경도 좌표 변환
- API 호출 실패 시 Retry 처리

## 2. 약국 추천

### 2.1 Haversine 거리 기반 추천
- 사용자 좌표로부터 Haversine 공식을 사용하여 거리 계산
- 반경 10km 내 약국 필터링
- 거리순 정렬 후 최근접 3곳 추천

### 2.2 카카오 카테고리 검색 기반 추천
- 카카오 카테고리 검색 API를 통한 주변 약국 조회
- Haversine 결과와 병합하여 최적 추천 제공

## 3. 방향 링크

### 3.1 Base62 단축 URL 생성
- 추천 약국별 고유 Base62 인코딩 URL 생성
- 단축 URL 접근 시 카카오맵 길찾기 페이지로 리다이렉트
- 출발지(사용자 위치) → 도착지(약국) 경로 자동 설정

## 4. 데이터 초기화

### 4.1 CSV 데이터 로드
- 공공데이터 CSV 파일로부터 약국 정보 일괄 등록
- 약국명, 주소, 위도, 경도 정보 파싱 및 저장

## 5. Redis 캐시 관리

### 5.1 약국 데이터 캐싱
- 약국 전체 데이터 Redis 캐시 저장
- TTL: 24시간
- 캐시 만료 시 DB에서 자동 리로드
- 캐시 히트 시 DB 조회 생략으로 성능 향상
